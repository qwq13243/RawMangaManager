# RawMangaManager

RawMangaManager 是一个本地运行的漫画下载与管理工具。它支持从多个在线源站下载漫画章节，并在本地进行管理、浏览和阅读。此外，项目集成了 OCR（文字识别）、机器翻译和图像去字引擎，可以对下载的漫画进行自动化翻译处理。

## 主要功能

*   **漫画下载**：支持 KlManga、NicoManga、Rawkuma 等源站的搜索与下载。
*   **本地库管理**：使用 SQLite 数据库存储漫画信息，支持自定义标题、封面和排序。
*   **自动翻译**：内置处理管线，包含文字检测、OCR 识别、机器翻译和气泡去字回填。
*   **更新检测**：支持批量检查已收藏漫画的最新章节并自动下载。
*   **本地阅读**：内置简易阅读器，支持在原版和翻译版之间切换。

## 安装与启动

### 环境要求
*   Windows 10/11 操作系统
*   Python 3.10 或更高版本（不支持3.14）

### 安装步骤
1.  下载或克隆本项目代码。
2.  运行根目录下的 `01_环境安装.bat` 脚本。
    *   脚本会自动配置国内镜像源，安装 PyTorch、Playwright 等必要依赖库，并下载浏览器内核。

### 启动
*   双击运行 `02_启动程序.bat`（或在命令行执行 `python main.py`）。

## 使用指南

### 1. 新建漫画
1.  在主界面右上角点击 **"➕ 新建漫画"**。
2.  输入漫画关键词（建议使用日文原名或英文名）并回车搜索。
3.  在搜索结果中选中目标漫画，可以预览章节数。
4.  点击 **"确认添加"**，程序会将漫画信息及章节列表存入本地数据库。

### 2. 下载章节
1.  在主界面点击任意漫画卡片，进入**漫画详情页**。
2.  **单章/选集下载**：在章节列表中勾选需要下载的章节，点击 **"下载选中"**。
3.  **批量下载**：点击 **"一键下载未下载"**，程序会自动下载所有状态为“未下载”的章节到最新一个已下载的章节。
4.  下载进度会在主界面底部的状态栏显示。

### 3. 检查更新
该功能用于同步源站的最新章节列表到本地数据库（不下载图片）：
1.  点击主界面顶部的 **"🔄 检查更新"** 按钮。
2.  程序会遍历所有“正在追更”的漫画，访问源站获取最新章节列表。
3.  如果有新章节，它们会以“未下载”状态出现在数据库中。
4.  建议在“一键追更”前先运行此功能。

### 4. 一键追更
该功能用于批量下载库中尚未下载的章节：
1.  点击主界面顶部的 **"⚡ 一键追更"** 按钮。
2.  程序会查找所有标记为“正在追更”且存在“未下载”章节的漫画。
3.  自动将这些章节加入下载队列并开始下载。
4.  下载完成后，如果配置了翻译引擎，会自动进入翻译队列。

### 5. 翻译与去字
本项目包含一个完整的翻译引擎，用于将生肉漫画转换为熟肉。

**配置翻译引擎：**
1.  点击主界面右上角的 **"⚙️ 设置"**。
2.  在 **"Saber 翻译设置"** 选项卡中配置：
    *   **翻译服务**：测试版本仅支持硅基流动的api与本地的sakurallm，其余配置请自行尝试。填写`API Key` 和 `Model Name`。
    *   **去字 (Inpainting)**：勾选启用后，程序会尝试擦除原气泡文字并填补背景。

**执行翻译：**
*   **自动翻译**：在“一键追更”流程中，下载完成的章节会自动进入翻译队列。
*   **手动翻译**：在漫画详情页，选中已下载的章节，点击 **"翻译选中"**。
*   **启动机翻**：在漫画详情页，点击 **"启动机翻"**，程序会自动翻译所有状态为“未翻译”的章节到最新一个已翻译的章节。

### 6. 阅读与浏览
*   **打开目录**：在漫画详情页点击 **"📂 打开目录"**，可直接在文件资源管理器中查看图片文件。
*   **本地阅读**：双击已下载的章节（或点击阅读按钮），会打开内置阅读器。
    *   阅读器支持翻页、选页、缩放。
    *   如果该章节已翻译，可以通过中键切换查看“原图”或“译图”。
*   **排序管理**：在主界面，您可以通过长按拖拽漫画卡片来调整顺序，也可以通过右键点击漫画卡片来调整。

## 常见问题

*   **应用程序报错**：如果应用程序报错，请及时反馈，或者使用源码部署的方式。
*   **OCR/翻译速度慢**：图像处理是计算密集型任务。目前未测试显卡加速功能，因此使用cpu处理单页可能需要数秒至数十秒。
*   **源站连接超时**：部分源站可能在特定网络环境下无法访问，请检查您的网络设置或者配置代理。可通过设置中关闭“无头模式”检查访问情况。
